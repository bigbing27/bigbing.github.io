发表于 2019-12-12 | 分类于 PHP

学习PHP的准备工作

windows下安装Apache+PHP+MySQL，即wamp

一、wamp、Lamp、LNMP

wamp:windows+Apache+mysql+php

Lamp:Linux+apache+mysql+php

lnmp:Linux+Nginx+mysql+php

综上，网站的服务器组成基本有操作系统+服务器+数据库+服务器脚本语言

1.操作系统选择：个人使用玩一玩一般是windows，实际生产使用中多为linux

2.服务器：一般是apache或者nginx

3.数据库：mysql是学习时使用最多的数据库，oracle多用于大型数据

4.服务器脚本语言：java php c++都可以

二、wamp

在学习中，不建议使用wamp server集成环境，虽然可以一键安装，但这样并不能使初学者理解他们之间的关系，不利于学习。

在服务器工作过程中，windows即为操作系统，apache、mysql、php就是安在电脑上的软件。apache是服务器与客户端交互的门户，客户端例如浏览器，apache接收来自客户端的请求，解析请求，下发指令，并返回请求的内容。apahce只能解析静态页面，而动态页面需要专用的脚本语言处理，这就用到了php语言。当浏览器请求动态页面时，apache会将内容交给php处理再返回。数据库用于存储数据，通过php脚本语言来调用。
