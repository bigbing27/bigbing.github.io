建站技术

  • hexo

    hexo基于Node.js，具有超快的生成速度，能让上百个页面在几秒内瞬间完成渲染；

    只需一条指令即可部署到github pages，heroku或其他平台；

    支持Markdown所有功能，甚至可以整合Octopress的大多数插件；

    强大的API带来无限可能，与数种模板引擎（EJS，Pug，Nunjucks）和工具（babel，PortCSS，Lass/Sass)轻易集成。

  • Github Pages

    Github Pages是Github免费给开发者提供的一款托管个人网站的产品，现在只能托管静态内容，非常适合用来搭建博客，不需要再另外买服务器，并且步骤十分简单。

  • Next主题

    坚持将复杂的细节隐藏，提供尽量少并且简便的设置，保持最大限度的易用性；

    除了MD支持的语法之外，借助Hexo提供的标签特性，为用户提供在书写文档时快速插入带特殊样式的内容；

    支持多种第三方服务来扩展站点内容。
